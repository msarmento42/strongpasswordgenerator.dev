Password Security Tips

Use Unique Passwords

Never reuse passwords across different accounts. If one gets breached, all your accounts remain safe.

Use a Password Manager

Password managers generate and store unique passwords so you only need to remember one master password.

Enable 2FA

Two-factor authentication adds an extra layer of security beyond just your password.

Longer is Stronger

A 20-character password with just letters is often stronger than a 12-character one with symbols.

Frequently Asked Questions

Are the passwords generated truly secure?

Yes! We use the Web Crypto API's cryptographically secure random number generator (CSPRNG) via crypto.getRandomValues(). This is the same standard used by password managers and security professionals.

Is my password sent to any server?

No. All password generation happens locally in your browser. Nothing is ever transmitted over the internet.

How is crack time calculated?

We calculate based on a brute-force attack scenario assuming 10 billion guesses per second (modern GPU cluster). Actual crack time varies based on attack method.

What is password entropy?

Entropy measures password randomness in bits. Higher entropy means more difficult to crack. 60+ bits is considered strong.

Is the Password Health Dashboard secure?

All data is stored locally in your browser using localStorage. Nothing is sent to any server. However, for production use, we recommend a dedicated password manager.
